Module Dockerfile_opam_cmdliner

module Dockerfile_opam_cmdliner: sig .. end
Command-line helper functions for scripts using Dockerfile_opam
val cmd : name:string ->
       version:string ->
       summary:string ->
       manual:string ->
       default_dir:string ->
       generate:(string -> 'a) -> 'a Cmdliner.Term.t * Cmdliner.Term.info
This generate a Cmdliner term with various defaults filled in for the manual page, and a default term that specifies an output directory.
val run : 'a Cmdliner.Term.t * Cmdliner.Term.info -> 'b
run executes the term that results from Dockerfile_opam_cmdliner.cmd and returns with a non-zero exit code if there is an error.
